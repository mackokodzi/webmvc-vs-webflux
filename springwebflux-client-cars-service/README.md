### Mongo
Install locally mongo:
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/#install

```
brew install mongodb-community@4.2
brew services start mongodb-community@4.2
```

open mongo:
```
mongo
```

find cars documents:
```
db.Car.find()
```
